# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/persona-bot.git

cd persona-bot
```

---

## Install Dependencies

### Frontend

```bash
cd client

npm install
```

### Backend

```bash
cd server

npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend.

```env
OPENAI_API_KEY=your_api_key

```

---

# ▶️ Run Locally

## Start Backend

```bash
cd server

npm run dev
```

## Start Frontend

```bash
cd client

npm run dev
```
